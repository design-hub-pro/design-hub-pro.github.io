# CalmClass: AI-Powered Emotional Wellness Monitoring System for K-12 Education

# CalmClass: 面向K-12教育的AI情绪健康监测系统

## Subtitle / 副标题

**Discrete Stress Detection and Intervention Platform for Neurodivergent-Inclusive Classrooms**

**面向神经多样性包容课堂的隐蔽压力检测与干预平台**

---

## Abstract / 摘要

### English

CalmClass presents an innovative AI-powered emotional wellness monitoring system designed to facilitate discrete communication between teachers and students in K-12 classroom environments. The system integrates a multi-modal sensing approach combining a portable desk sensor pad and optional wearable devices to capture real-time stress indicators including tactile interaction patterns (tapping frequency, fidget behaviors), biometric signals, and behavioral analytics. A proprietary machine learning pipeline processes these signals to identify early signs of overstimulation and emotional overload. The system features a student-facing "Safety Plan" activation button enabling private, stigma-free requests for assistance, while a comprehensive teacher dashboard provides actionable insights with contextual recommendations such as "Student may be overwhelmed → offer break option." Additionally, the platform includes a curated library of sensory-friendly activities and digital fidget tools. Preliminary evaluations demonstrate significant improvements in early intervention rates and student emotional regulation outcomes, positioning CalmClass as a transformative tool for creating inclusive, responsive learning environments.

### 中文

CalmClass是一款创新的AI驱动情绪健康监测系统，专为促进K-12课堂环境中教师与学生之间的隐蔽沟通而设计。该系统集成了多模态传感方法，结合便携式桌面传感垫和可选可穿戴设备，捕捉实时压力指标，包括触觉交互模式（敲击频率、坐立不安行为）、生物特征信号和行为分析。专有的机器学习管道处理这些信号，以识别过度刺激和情绪超载的早期迹象。系统配备学生端"安全计划"激活按钮，实现私密、无污名化的求助请求，同时全面的教师仪表板提供可操作的洞察和情境化建议，如"该学生可能感到不适→建议提供休息选项"。此外，平台还包含精选的感官友好型活动和数字减压工具库。初步评估显示，该系统在早期干预率和学生情绪调节效果方面均有显著改善，使CalmClass成为创建包容性、响应式学习环境的变革性工具。
