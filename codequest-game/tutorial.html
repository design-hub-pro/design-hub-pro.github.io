Move Your Hero

Use simple commands to navigate your hero through the maze:

UP

DOWN
Move down
LEFT

RIGHT

ROTATE

2

Write a Program

Type commands line by line. Use REPEAT to loop:

RIGHT
RIGHT
REPEAT 3 [DOWN]
LEFT
Executes: RIGHT → RIGHT → DOWN → DOWN → DOWN → LEFT
