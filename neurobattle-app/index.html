Abstract / 摘要

NeuroBattle Arena presents an innovative competitive gaming platform where users upload custom neural network-powered agents to compete in strategic turn-based combat. The platform emphasizes accessibility through a streamlined game mechanic featuring five core actions: Attack, Defend, Heal, Charge, and Special. Users train their neural networks locally using provided SDKs and upload agent code with optional pretrained weights. Each agent receives a compact state vector containing health points, energy levels, buff/debuff status, and recent opponent action history, outputting probabilistic action selections through their neural architecture. The platform executes deterministic match simulations with detailed battle logs, replay visualizations, and performance analytics. A global leaderboard with ELO-based rankings drives competitive engagement.

NeuroBattle Arena是一个创新的竞技游戏平台，用户可以上传自定义的神经网络驱动智能体参与策略性回合制对战。平台通过简化的游戏机制强调可访问性，包含五种核心动作：攻击、防御、治疗、蓄力和特殊技能。用户使用提供的SDK在本地训练神经网络，并上传包含可选预训练权重的智能体代码。每个智能体接收紧凑的状态向量，通过其神经架构输出概率性动作选择。平台执行确定性对战模拟，提供详细的战斗日志、回放可视化和性能分析。全球排行榜系统采用ELO评分排名来推动竞技参与。
